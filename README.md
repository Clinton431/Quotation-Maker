# Quotation Maker

Professional quotation generator built with MERN stack.

## Features

- Modern, responsive UI
- PDF generation
- MongoDB database
- Real-time preview
- Professional output

## Quick Start

1. Install dependencies:

```bash
npm install
cd client && npm install
cd ..
```

2. Start the app:

```bash
npm run dev
```

3. Open http://localhost:3000

## Tech Stack

- React 18
- Node.js + Express
- MongoDB
- Tailwind CSS
- PDF Generation (html2canvas + jsPDF)
